# Resource-A-Tron

A comprehensive team management platform for insurance advisory teams to manage resources, track time, schedule activities, and monitor team performance.

## Features

- User management with skills tracking
- Time sheets and time tracking
- Activity management (projects, internal activities, PTO)
- Resource assignment with intelligent recommendations
- Program management
- Activity scheduling
- Utilization visualization
- Team KPIs and progress tracking
- Role management with billing rates
- Project scoping and financial calculations

## Tech Stack

- **Frontend**: React 18+ with TypeScript, Tailwind CSS, Vite
- **Backend**: Node.js with Express/NestJS, TypeScript
- **Database**: PostgreSQL
- **Hosting**: Railway
- **CI/CD**: GitHub Actions

## Getting Started

### Prerequisites

- Node.js 18+
- PostgreSQL 14+
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables (see `.env.example` files in frontend and backend)

4. Run development servers:
   ```bash
   npm run dev
   ```
